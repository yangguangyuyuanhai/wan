## 🎯 异步版本特性

### 核心优势
- ✅ 支持多相机并发采集
- ✅ 基于 asyncio 的异步处理
- ✅ 高性能并发管道
- ✅ 自动负载均衡
- ✅ 非阻塞I/O操作

### 适用场景
- 多相机同时采集
- 高并发图像处理
- 实时性要求高的场景
- 需要充分利用多核CPU
