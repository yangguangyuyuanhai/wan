## 🚀 快速启动

```bash
# 进入Qt版本目录
cd service_qt

# 运行Qt GUI
python run_gui.py

# 或使用批处理文件（Windows）
启动GUI.bat
```

## 📦 依赖要求

```bash
# 安装Qt依赖
pip install PyQt5
pip install -r requirements_qt.txt
```
